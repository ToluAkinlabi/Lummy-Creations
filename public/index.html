Featured Work

Case Study: AICA - Abnormal AI Communications Assistant

AICA: Incident Response AI Platform

Challenge

Incident response teams needed an intelligent system to automate communication during security events while maintaining data security and contextual awareness across varying severity levels.

Solution

Designed and developed a comprehensive AI-powered communications platform featuring:

  • Advanced Guardrails: Regex-based redaction for IPs, emails, hostnames, and internal IDs; automatic stack-trace line removal
  • Intelligent Auto-Cadence: Severity-based communication scheduling (SEV1–SEV4)
  • Dynamic Prompt Engineering: Stage-aware prompt construction (initial/ongoing/resolution)
  • Multi-Format Export: StatusPage snippets and email body templates
  • LLM Integration: OpenAI-powered standard draft generation
