## Deployment

### Vercel (Recommended)

1. Install Vercel CLI:
```bash
npm i -g vercel
```

2. Deploy:
```bash
vercel
```

3. Follow prompts to link project and deploy

### Custom Domain Setup

After deployment, add your custom domain in the Vercel dashboard:
- Go to Project Settings → Domains
- Add your domain
- Update DNS records as instructed
